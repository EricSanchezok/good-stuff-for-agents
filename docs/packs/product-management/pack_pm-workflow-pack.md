# Product Management Workflow Pack

I made this pack as a prepared route for agents working in product management. Execute a complete product management workflow from user research through prioritized delivery planning

## When I’d reach for it

- An agent needs 1 help inside a product management task.
- An agent needs 2 help inside a product management task.
- An agent needs 3 help inside a product management task.

## The route I built

1. **Discover** — Synthesize raw user interview data into structured, evidence-backed findings I put discover-interview-synthesis here.
2. **Define** — Frame the problem space with hypothesis formation and formal problem statements I put define-hypothesis and define-problem-statement here.
3. **Prioritize** — Rank competing hypotheses using evidence-backed frameworks I put define-prioritization-framework here.
4. **Deliver** — Produce PRDs from framed problems, then decompose into sprint-ready user stories I put deliver-prd and deliver-user-stories here.
