# tiktok-ads

> Ready to use

## Summary

This is a TikTok advertising platform reference card that does one thing the other ad-platform skills in this collection don't: it actually teaches creative strategy. The TikTok Creative Formula section ("Hook → Value/Story → CTA") is concrete, operational, and tied to platform-specific behaviors that can't be derived from general marketing knowledge. But the rest of the skill is a feature catalog, and the gap between the creative section and everything else is wide enough to drive a campaign budget through.

## Public Analysis Summary

Pick this skill if you specifically need TikTok creative strategy guidance and already know how to run paid social campaigns — the creative formula and fatigue insights are genuine and platform-specific. Skip it for anything involving campaign setup, TikTok Shop, or policy navigation. The biggest risk is recommending a creative velocity the user can't sustain. The biggest benefit is the creative formula, which encodes 12-18 months of trial-and-error learning into seven bullet points.
