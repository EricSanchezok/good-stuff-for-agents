# Skill Page Template

A generated skill page must include:

1. generated banner and frontmatter;
2. summary and status;
3. source and license status;
4. capabilities and workflow stages;
5. best used for / not for;
6. inputs, outputs, and handoff outputs;
7. related packs;
8. related skills from relation edges;
9. public analysis summary;
10. confidence and limitations.

If analysis is missing, show `Analysis pending` rather than inventing facts.
