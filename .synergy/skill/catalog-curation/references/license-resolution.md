# License Resolution

When resolving license uncertainty:

1. inspect upstream license files and package metadata;
2. record SPDX identifier when known;
3. mark `verified: true` only with explicit evidence;
4. preserve evidence path or URL;
5. block raw content mirroring if license remains unknown or incompatible;
6. update public docs through publishing after catalog records change.

Unknown license may still allow metadata, hashes, provenance, short excerpts, and analysis when legally appropriate. Do not store third-party raw content without permission.
