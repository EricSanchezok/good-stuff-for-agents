# Source Quality Rubric

Source quality affects confidence and featured placement.

Signals:

- license verified and compatible;
- source is active and reachable;
- skill-like artifacts are parseable;
- upstream has stable paths or versions;
- source includes clear trigger/procedure/input/output information;
- source does not require secrets or private access for public cataloging;
- source updates can be tracked deterministically.

Unknown license does not block metadata cataloging, but it blocks raw mirroring and lowers confidence.
