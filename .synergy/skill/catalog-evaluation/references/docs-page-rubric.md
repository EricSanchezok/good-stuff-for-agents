# Docs Page Rubric

Generated public pages should be useful and traceable.

Check that each page has:

- generated banner;
- machine-readable frontmatter;
- source record path and hash metadata where applicable;
- clear status labels for preview, stale, deprecated, removed, or blocked content;
- no candidate-only pack content;
- links that resolve inside the repo;
- public summaries that avoid license-prohibited raw mirrors.

Docs quality is evaluated after rendering; schema validation still belongs to catalog-data.
