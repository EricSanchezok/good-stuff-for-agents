# Discovery Channels

Use public, non-invasive discovery channels:

- GitHub search for `SKILL.md`, `.synergy/skill`, `.claude/skills`, Codex skills, agent workflows.
- Official docs for agent platforms and MCP ecosystems.
- Public package registries and marketplaces when they expose skill-like metadata.
- Community repos only when source provenance and license can be recorded.

Record search query, URL, and reason for every accepted candidate.
