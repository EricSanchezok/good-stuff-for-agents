# Unity Animation

## Core Purpose
Guides agents through Unity's animation system — Animator Controller, animation state machines, blend trees, IK, and integration with Mecanim and Timeline.

## Trigger Semantics
Load when an agent implements character animations in Unity, designs animation state machines, sets up blend trees for movement blending, or uses Timeline for cinematic sequences.

## Capability Breakdown
Covers Animator Controller setup and state management, animation state machine design, blend trees for parameter-blended animation, Avatar masks for partial body animation, IK goals, Timeline integration, and animation events for gameplay callbacks.

## Workflow Role
Implementation — applied when bringing characters to life through animation.

## Inputs / Outputs
Inputs: character rig and animation clips. Outputs: Animator Controller, state machines, blend trees.

## Tool and Permission Profile
Unity editor (Animation window, Animator window). Standard script editing.

## Compatibility Notes
Complements unity-2d and unity-csharp-scripting for animation logic code.
