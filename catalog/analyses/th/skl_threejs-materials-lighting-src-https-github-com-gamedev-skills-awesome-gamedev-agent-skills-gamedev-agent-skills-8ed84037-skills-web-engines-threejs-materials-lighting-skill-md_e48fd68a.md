# Threejs Materials Lighting

## Core Purpose
Guides agents through Three.js materials and lighting — PBR materials (MeshStandardMaterial, MeshPhysicalMaterial), texture maps (normal, roughness, metallic), light types, and shadow configuration.

## Trigger Semantics
Load when an agent applies realistic materials in Three.js, configures PBR textures, sets up shadow-casting lights, or optimizes material performance.

## Capability Breakdown
Covers MeshStandardMaterial and MeshPhysicalMaterial properties, texture maps (albedo, normal, roughness, metallic, ao, emissive), light types and shadow mapping setup (shadow map size, bias, camera), and environment maps for reflections.

## Workflow Role
Implementation — applied when creating visually rich 3D scenes in Three.js.

## Inputs / Outputs
Inputs: visual quality requirements, texture assets. Outputs: material configs, lighting setup, shadow configuration.
