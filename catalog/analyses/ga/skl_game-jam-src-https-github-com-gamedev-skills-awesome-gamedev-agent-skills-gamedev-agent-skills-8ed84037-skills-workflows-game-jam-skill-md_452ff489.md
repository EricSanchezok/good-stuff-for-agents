# Game Jam

## Core Purpose
Guides agents through game jam workflows — rapid prototyping strategies, scope management for time constraints, minimal viable game design, team coordination, and submission preparation.

## Trigger Semantics
Load when an agent participates in a game jam, needs rapid game prototyping guidance, or optimizes development workflow for time-boxed game creation.

## Capability Breakdown
Covers jam-specific scope management (vertical slice prioritization), rapid prototyping techniques (placeholder assets, simple mechanics first), time boxing and milestone planning, toolchain optimization (template projects, asset packs), and submission packaging (build optimization, itch.io upload).

## Workflow Role
Planning/Implementation — applied during time-constrained game development events.

## Inputs / Outputs
Inputs: jam theme, time limit, team composition. Outputs: scoped game plan, prototype, submission-ready build.

## Tool and Permission Profile
Standard game engine tools. Itch.io or other platform account for submission.

## Compatibility Notes
Complements prototype-fast for rapid iteration and godot-export or unity-build-pipeline for quick builds.

## Conflict Notes
No conflicts.

Unique in the catalog — no other game jam workflow skill.

## Evaluation Hooks
Verify the skill covers theme interpretation techniques and time-boxed decision-making frameworks.
