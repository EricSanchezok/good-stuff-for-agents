# Pixijs Rendering

## Core Purpose
Guides agents through 2D rendering with PixiJS — DisplayObject and Container hierarchy, sprite and texture management, WebGL renderer configuration, filters, and performance optimization.

## Trigger Semantics
Load when an agent builds 2D rendering in the browser with PixiJS, manages display list hierarchies, applies WebGL filters, or optimizes sprite batching.

## Capability Breakdown
Covers Application setup and renderer selection, DisplayObject/Container hierarchy, Sprite and Texture management (from spritesheets, atlases), Graphics and text objects, filters (blur, color matrix, displacement), and rendering performance optimization (batching, pooling, texture atlas usage).

## Workflow Role
Implementation — applied when creating 2D interactive graphics or games using PixiJS in the browser.
